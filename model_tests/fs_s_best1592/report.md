## 7. 复现命令

在项目根目录执行：

```powershell
.\.venv-cpython\Scripts\python.exe model_tests\fs_s_best1592\detect_teacher_video.py
```

如需修改检测参数：

```powershell
.\.venv-cpython\Scripts\python.exe model_tests\fs_s_best1592\detect_teacher_video.py `
  --conf 0.5 --imgsz 1088 --frame-stride 20 --device cpu
```

默认模型为 `models/fs_s_best1592.pt`，默认视频为 `data/1-1.01_74_fs.mp4`。当前项目未保存原始视频，请先将视频放入该位置，或在命令中通过 `--video "视频完整路径"` 指定。`--frame-stride 20` 表示每秒取 1 帧；若改为 `1`，会检测原视频全部 24,600 帧，但运行时间和输出体积都会显著增加。

## 8. 实现说明

检测脚本为 [detect_teacher_video.py](detect_teacher_video.py)，流程与 `test_model.py` 一致：使用 Ultralytics `YOLO` 加载权重、指定置信度阈值和设备完成推理，并额外针对真实视频实现了抽帧、标注视频写入、CSV 明细与 JSON 汇总。

## 9. 结论与限制

- 指定模型能够在整段视频的每个抽样时间点检测到至少一个目标，器材和手部等主要类别均有检出。
- `水柱` 在当前阈值和抽样策略下未检出；二氧化锰固体、燃烧木条的检出次数较少，建议在对应实验片段逐帧人工复核。
- 本结果用于展示模型在该视频上的检测输出，不构成准确率验证报告。
- 当前环境为 CPU。若后续恢复可用的 CUDA PyTorch 环境，可将 `--device cuda` 用于加速。
