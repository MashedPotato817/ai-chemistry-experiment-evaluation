# 智能化学实验熟练度评估系统

## 系统定位
基于计算机视觉 + 大模型的初中化学实验实时评估与指导平台

---

## 一、计算机视觉感知模块
> 看懂实验：识别器材、理解画面

### 目标检测
- 模型架构：YOLOv11s（轻量化定制）
- 训练数据：导师自制数据集 fs_new，训练 1200 轮
- 识别目标：19 类实验器材与关键状态
  - 器材类：锥形瓶、集气瓶、酒精灯、长颈漏斗、止水夹、导管等
  - 状态类：气泡、水柱、燃烧木条、熄灭木条
- 推理性能：65.6 FPS（1088×1088），每秒处理约 48 帧完整流程

### 置信度控制
- 算法：阈值过滤
- 推荐阈值：0.5（兼顾精度与召回）
- 支持 CPU / GPU 双端自动切换

## 二、实验行为时序分析模块
> 看懂过程：判断步骤、识别动作、发现异常

### 步骤自动推断
- 算法：STEP_INDICATORS 规则匹配 + 得分排序
- 逻辑：必要器材全部出现 → 计算匹配得分 → 取最高分步骤
- 覆盖 9 个标准步骤：准备器材 → 添加药品 → 组装装置 → 检查气密性 → 点燃酒精灯 → 收集气体 → 验证气体 → 熄灭酒精灯 → 整理器材

### 运动与动作分析
- 算法：物体追踪（deque 滑动窗口）+ 位移向量计算
- 功能：检测手部移动方向、速度、停顿状态
- 动作分类：操作锥形瓶、安装橡胶塞、连接导管、收集气体等 18 类

### 安全事件检测
- 算法：欧氏距离计算
- 规则：手部与酒精灯/火柴/燃烧木条距离 < 60px → 触发安全预警
- 实时输出预警信息

### 器材完整性检查
- 算法：集合比对
- 功能：自动统计已检测器材，输出缺失器材列表

## 三、熟练度量化评分模块
> 量化水平：客观评分、百分位排名

### S1 时长得分（权重 70%）
- 算法：正态分布累积函数
- 公式：`S1 = 2 - 2Φ((μ - t) / σ)`
- 参数：μ=480s（均值），σ=90s（标准差），随数据积累动态更新
- 完成越快 → S1 越高（最高 2.0）

### S2 表现得分（权重 30%）
- 算法：多维加权扣分模型
- 输入维度：
  - 步骤完成率（乘数系数 0.4~1.0）
  - 操作质量（大模型评估，0~1）
  - 错误次数（每次扣 0.1，上限 1.0）
  - 提问次数（超过 2 次开始扣分）
  - 安全事件次数（每次扣 0.1）
  - 安全规范违规（上限压至 1.2）

### 最终得分与排名
- 公式：`S = 0.7 × S1 + 0.3 × S2`（满分 2.0）
- 等级划分：优秀（≥1.8）/ 良好（≥1.4）/ 合格（≥1.0）/ 需改进
- 百分位排名：基于得分正态分布估计，输出"超越 X% 的学生"
- 统计参数更新：Welford 在线算法，样本 ≥5 条自动更新 μ、σ

## 四、大模型智能交互模块
> 实时指导：纠错、提示、个性化反馈

### 多平台接入
- 接口标准：OpenAI 兼容接口
- 支持服务商：Kimi（moonshot-v1-128k）、通义千问、DeepSeek、OpenAI
- 切换方式：仅修改 .env 三行配置，零代码侵入

### 实时指导生成
- 输入上下文：当前步骤 + 检测到的器材 + 错误次数 + 已用时间
- 输出：自然语言操作指导（如"使用滴管缓慢加入双氧水至锥形瓶，注意不要溢出"）
- 降级机制：API 不可用时自动切换内置规则响应，保障系统可用性

## 五、系统集成与工程化模块
> 跑起来：用户管理、数据存储、界面交互

### 用户管理
- 算法：bcrypt 密码哈希 + JWT 令牌认证
- 功能：注册、登录、会话管理、权限控制

### 数据存储
- 技术：SQLite + SQLAlchemy ORM
- 存储内容：用户信息、实验记录、S1/S2/最终得分、百分位排名、实验时长

### 交互界面
- 框架：Streamlit
- 功能页面：登录注册 / 实验主页 / 实时检测 / 步骤导航 / 评分报告 / 历史记录

### 部署方案
- 支持环境：Python 3.9+，CPU / GPU 自动切换
- 启动方式：`python run.py` 一键启动
- 日志监控：结构化日志，自动写入 logs/ 目录

## 六、实验报告模块
> 记录成果：自动生成评估报告，支持历史回溯

### 在线报告展示（已实现）
- 最终得分 / 等级 / 百分位排名
- S1 时长得分 + S2 表现得分明细
- 操作错误次数、提问次数、安全事件统计
- 完成步骤数 / 总步骤数
- 个性化改进建议（基于规则自动生成）
  - 错误多 → 提示加强基本操作练习
  - 安全事件 → 提示注意实验安全规范
  - 提问多 → 建议课前充分预习
  - 用时长 → 建议熟悉操作流程

### 历史记录查询（已实现）
- 按实验类型、日期筛选历史记录
- 展示每次实验的 S1 / S2 / 最终得分 / 用时
- 数据持久化存储（SQLite），支持长期追踪

### PDF 报告导出（规划中）
- 一键导出完整实验评估报告
- 包含实验过程关键截图与评分详情
